# Code of Conduct

`vla_zoo` is a robotics infrastructure project. We want technical disagreement to be direct, useful, and respectful.

## Expected Behavior

- Be clear and specific when reporting problems.
- Assume robotics deployments have real safety constraints.
- Give credit to upstream VLA, robotics, and benchmark projects.
- Keep criticism focused on code, design, documentation, and reproducible behavior.
- Help newcomers understand the runtime boundary before proposing large scope changes.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks or repeated hostile behavior.
- Publishing private information without permission.
- Encouraging unsafe robot operation or bypassing safety constraints.
- Misrepresenting model performance, benchmarks, or hardware readiness.

## Reporting

If you need to report a conduct issue, open a private security advisory if privacy matters, or contact the maintainers through the repository owner. Maintainers may remove comments, close issues, or restrict participation when behavior harms the project.
